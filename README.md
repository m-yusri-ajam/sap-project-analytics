# SAP Project Analytics & Governance 🏗️

This repository demonstrates the architectural logic used to manage complex infrastructure portfolios. It focuses on the transition from **SAP ERP (PS/MM)** raw exports to executive-level decision support.

### 📈 Core Methodologies
- **WBS Hierarchy Management:** Logic for tracking CAPEX/OPEX across multi-level Work Breakdown Structures.
- **ETL Pipelines:** SQL queries and Power Query (M) scripts for cleaning "dirty" ERP data.
- **Financial Modeling:** Integration of FMVA® principles for ROI and Variance analysis.

### 📂 Repository Structure
- `/sql`: Data cleaning and aggregation scripts.
- `/docs`: Functional specifications for SAP lifecycle re-engineering.
- `/templates`: Power BI (PBIT) file schemas for executive dashboards.
